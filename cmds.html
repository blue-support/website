Bot Commands

/help – Zeigt alle Befehle an
/setup-globalchat – Setzt einen Channel als Globalchat. (Admin only)
/remove-globalchat – Entfernt einen Globalchat-Channel. (Admin only)
/global-dm-active – Aktiviere oder deaktiviere DM-Benachrichtigungen im Globalchat.
/bot-invite – Bot einladen
/tictactoe – Starte ein Tic Tac Toe Spiel
/setup-counting – Setzt den Kanal für das Zählspiel. (Admin only)
/setup-errate-zahl – Startet ein Errate-die-Zahl-Spiel in einem Kanal. (Admin only)
/join – Bot joint deinem Sprachkanal (Admin only)
/leave – Bot verlässt den Sprachkanal (Admin only)
/aktueller-song – Zeigt den aktuell laufenden Song an
/vorheriger-song – Zeigt den Song an, der davor abgespielt wurde
/list-songs – Zeigt eine Liste mit allen Songs an
/kiss – Küsse deinen Crush 💋
/ship – Finde heraus wie gut zwei Leute zusammenpassen 💖
/ban – Bannt einen User mit Grund und optionaler Dauer. (Admin only)
/unban – Entbannt einen User manuell. (Admin only)
/kick – Kickt einen User mit Grund. (Admin only)
/mute – Muted (Timeout) einen User für eine bestimmte Dauer. (Admin only)
/unmute – Entmutet (Timeout Ende) einen User manuell. (Admin only)
